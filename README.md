# AI Medical Report Explainer

AI-powered healthcare platform that converts complex medical reports into patient-friendly explanations using OCR, NLP, RAG, and LLMs.

---

# Features

- Medical PDF upload
- OCR extraction
- AI-generated summaries
- Abnormality highlighting
- Medical terminology explanations
- Risk indicators
- Downloadable reports
- Healthcare RAG pipeline

---

# Tech Stack

## Frontend
- Next.js
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

## AI
- OpenAI
- LangChain
- ChromaDB

## OCR
- Tesseract OCR
- OpenCV
- pdfplumber
